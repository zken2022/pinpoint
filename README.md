图钉 (Pinpoint) - 图像地理位置智能分析系统
【让像素，归于经纬】
1. 产品概述
图钉
是一款基于人工智能的图像地理位置分析系统，能够通过分析图像内容和提取EXIF数据，智能推断图像拍摄的地理位置，并在地图上精确标注。

智能分析：利用先进的AI模型分析图像内容，推断地理位置信息
EXIF提取：自动提取图像中的EXIF元数据，获取原始拍摄位置
地图可视化：在交互式地图上直观显示分析结果
历史记录：保存分析历史，方便用户回顾和管理
报告生成：自动生成详细的分析报告，包含位置信息和分析依据
2. 核心功能
2.1 图像上传与处理
支持多种图像格式：JPG、PNG等常见格式
拖拽上传：支持文件拖拽直接上传
EXIF自动提取：上传后自动提取图像的EXIF元数据，包括拍摄时间、设备信息和地理位置（如果有）
2.2 AI智能分析
多模型支持：支持多种AI模型进行图像分析
实时分析：提供分析过程的实时反馈
智能推断：基于图像内容推断拍摄地点，包括地标、建筑、自然景观等
分析依据展示：显示AI分析的思考过程，增强结果可信度
2.3 地图可视化
双模式定位：支持AI推理定位和EXIF原始定位两种模式
高精度地图：使用Leaflet地图库，提供高精度地理信息
交互操作：支持地图缩放、平移等交互操作
位置标记：在地图上精确标记分析出的地理位置
2.4 分析报告
详细报告：生成包含地理位置、分析依据、置信度等信息的详细报告
导出功能：支持报告导出和分享
数据可视化：通过图表和地图直观展示分析结果
2.5 历史记录管理
自动保存：自动保存分析历史，最多保存最近5条记录
快速访问：点击历史记录可快速查看之前的分析结果
管理功能：支持删除单条记录或清空全部历史
3. 技术架构
3.1 前端技术栈
React 19.0.0：构建用户界面
TypeScript：提供类型安全
Tailwind CSS：实现响应式设计和现代UI
Leaflet：提供地图可视化功能
Lucide React：提供图标支持
Motion React：实现流畅的动画效果
3.2 后端服务
AI服务：集成硅基流动 API和其他AI模型
数据处理：处理图像上传和EXIF数据提取
本地存储：使用localStorage存储用户配置和分析历史
3.3 核心流程
图像上传：用户上传图像，系统提取EXIF数据
AI分析：调用AI模型分析图像内容，推断地理位置
结果展示：在地图上显示分析结果，生成详细报告
历史保存：将分析结果保存到本地历史记录
4. 应用场景
4.1 旅行记忆管理
自动地理标记：为旅行照片自动添加地理位置标签
旅行路线重建：根据照片的地理位置信息重建旅行路线
地点回忆：通过照片快速回忆曾经去过的地方
4.2 新闻媒体应用
事件地点确认：确认新闻图片的拍摄地点
媒体素材管理：为媒体素材添加地理位置元数据
新闻真实性验证：验证新闻图片的拍摄地点是否与报道一致
4.3 调查取证
图像来源分析：分析图像的拍摄地点
证据链构建：通过地理位置信息构建完整的证据链
场景还原：根据图像内容和地理位置还原事件场景
4.4 社交媒体
智能地理标记：为社交媒体图片自动添加地理位置标签
地点分享：方便用户分享去过的有趣地点
旅行足迹：记录用户的旅行足迹，生成可视化地图
